## 🖼️ GLASSMORPHISM FRAMEWORK

### **The 5-15 Rule**

```
┌─────────────────────────────────────────────────────────────────────────────┐
│ BACKGROUND TINT:    15% color opacity                                       │
│ BORDER HIGHLIGHT:    8% white opacity                                       │
│ BACKDROP BLUR:       20px gaussian blur                                     │
│ SHADOW DEPTH:        Multiple layers (8px, 32px, 80px)                     │
│ INSET GLOW:          0 0 60px at 3% opacity                                 │
└─────────────────────────────────────────────────────────────────────────────┘

FORMULA FOR ANY PATHWAY COLOR:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
background: linear-gradient(135deg,
    rgba(R, G, B, 0.15) 0%,
    rgba(R, G, B, 0.1) 100%
);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.08);
box-shadow:
    0 8px 32px rgba(R, G, B, 0.15),
    0 0 80px rgba(R, G, B, 0.05),
    inset 0 0 60px rgba(R, G, B, 0.03);
```
